# Winbar 버퍼 표시 기능

## 개요
각 윈도우 상단에 현재 탭의 버퍼 목록을 표시하여 빠른 확인과 전환 가능

## 표시 형식
```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
 1:index.js │ [2:App.jsx ●] │ 3:styles.css
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```
- `[현재 버퍼]` - 대괄호로 강조 표시
- `●` - 수정된 버퍼 표시
- `1:` - 버퍼 인덱스 (빠른 전환용)

## 키 매핑

### 숫자 단축키 (NEW!)
- `,1` ~ `,9` - 해당 번호의 버퍼로 즉시 전환
- 예: `,3` = 3번째 버퍼로 이동

### 기존 버퍼 네비게이션
- `,h` - 이전 버퍼
- `,l` - 다음 버퍼  
- `,bd` - 버퍼 삭제
- `,bb` - fzf 버퍼 목록 (검색용)

### 탭 네비게이션
- `tn` - 새 탭
- `td` - 탭 닫기
- `th` / `tl` - 이전/다음 탭

## 특징
✅ **항상 표시** - 매번 `,bb` 입력 불필요
✅ **탭별 독립** - 각 탭의 버퍼만 표시
✅ **빠른 전환** - 숫자키로 즉시 이동
✅ **시각적 피드백** - 현재/수정 상태 구분
✅ **마우스 지원** - 클릭으로도 전환 가능

## 색상 테마
Solarized Light 테마와 조화되는 색상:
- 현재 버퍼: 진한 배경
- 수정된 버퍼: 빨간색 점(●)
- 버퍼 번호: 파란색

## 사용 시나리오

### 시나리오 1: 빠른 파일 전환
```
작업 중 → 상단 확인 → ,3 입력 → 3번 버퍼로 이동
```

### 시나리오 2: 수정 파일 확인
```
● 표시 확인 → 저장 필요한 파일 파악
```

### 시나리오 3: 탭별 작업 분리
```
탭1: [컴포넌트 3개]
탭2: [API 파일 2개]
탭3: [테스트 파일 4개]
```

## 명령어
- `:WinbarRefresh` - 버퍼 목록 강제 새로고침

## 자동 업데이트 시점
- 버퍼 진입/생성
- 파일 저장/수정
- 탭 전환
- 버퍼 삭제
